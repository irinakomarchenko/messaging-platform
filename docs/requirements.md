# Requirements

## Functional Requirements

1. One-to-one text messaging.
2. Group chats.
3. Named channels.
4. Online users receive messages in real time.
5. Offline users receive missed messages after reconnect.
6. Only text messages are supported in the first version.
7. Users have visible ONLINE / OFFLINE status.

## Out of Scope

1. Images, videos and files.
2. Video calls.
3. Typing indicator.
4. Message editing and deletion.
5. End-to-end encryption.
